## Název projektu

 - Mini atlas domácích rostlin

## Žák

 - Rudolf Selinka

## Stručný popis tématu (3–5 vět)

 - Jednoduchý web o pokojových rostlinách. Uživatel si může prohlédnout různé rostliny, jejich fotky a základní informace.
 - Data se budou načítat z JSON souboru. Web bude mít několik základních sekcí. Cílem je ukázat jednoduchou práci s daty a webovým designem.

## Cílová skupina

 - Lidé, kteří mají doma rostliny.
 - Začátečníci, kteří se chtějí naučit, jak o rostliny pečovat.

## Návrh obsahových sekcí

 - Úvod – krátké vysvětlení, o čem web je.
 - Rostliny – seznam rostlin z JSON (foto, název, náročnost, zalévání).
 - Rady – jednoduché tipy pro péči o rostliny.
 - Kontakt – jednoduchý formulář + e-mail.

## Návrh designu / wireframe

 - Světlé barvy (hlavně zelená a bílá).
 - Jednoduché karty rostlin v gridu.
 - Velké obrázky, méně textu.
 - Jednoduché a přehledné rozložení.

## Použité technologie

 - HTML, CSS
 - JavaScript
 - JSON soubory (data)
 - AJAX pro načítání dat
 - PHP pro odeslání formuláře
 - Git + GitHub

## Očekávaný přínos / co se naučím

 - práce s JSON
 - načítání dat pomocí AJAX
 - tvorba jednoduchého responzivního webu
 - základy práce s formuláři
 - práce s GitHubem
